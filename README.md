Aplikacja webowa FAQ, do zarządzania bazą wiedzy.

Główne funkcjonalności:
1. System kategorii i podkategorii.

2. Wyszukiwanie pytań oraz filtrowanie kategoriami.

3. Zaawansowane role: 
- Użytkownik niezalogowany - tylko przeglądanie pytań i odpowiedzi.
- Użytkownik - zgłaszanie pytań i odpowiedzi.
- Moderator - udzielanie odpowiedzi, edycja treści.
- Administrator - zarządzanie kategoriami i uprawnieniami użtkowników.

4. Panel administratora z raportami (kategorie z największą liczbą pytań bez odpowiedzi, najaktywniejsi odpowiadający).

5. System zatwierdzania treści, pytanie musi zostać zaakceptowane przez moderatora lub admina, zanim stanie się publiczne.
